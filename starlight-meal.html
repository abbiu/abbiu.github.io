Starlight Meal Optimizer

Optimization model for planning school meals under cost & nutrition constraints.
Optimization Python · Gurobi / OR-Tools Operations & planning

Problem

A school needs to plan weekly meal combinations that meet nutrition requirements, stay within budget, and avoid overloading kitchen capacity — all while keeping students reasonably happy (no, they cannot eat fries every day).

Approach

  • Defined decision variables for how many servings of each meal to serve per day.
  • Set up constraints for budget, nutrition, kitchen capacity, and minimum variety.
  • Built an objective function to minimize total cost while meeting all requirements.
  • Implemented and solved the model in Python using a linear optimizer.

What I contributed

  • Translated a fuzzy real-life problem into a clear mathematical model.
  • Structured the data and model so it’s easy to tweak scenarios and constraints.
  • Created simple outputs & visualizations to explain trade-offs to non-technical audiences.

Results & impact

The model produced meal plans that met all constraints and showed how cost and variety change when adjusting budget or nutrition requirements. This structure could be adapted for menu planning, workforce scheduling, or other planning use cases.

Tech stack

Python, linear optimization library (e.g., Gurobi / OR-Tools), pandas, basic visualization.
